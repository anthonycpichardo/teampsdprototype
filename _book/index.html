    • 3.1 Day 1: Design Thinking
      • 3.1.1 1. Empathize - To gain an empathic understanding of the need or problem you are trying to solve in terms of people, processes that set aside individual assumptions to get insight into the need.
      • 3.1.2 2. Define the problem - This is when we synthesize observations into holistic point of view (POV). Goal is to make linkages, so that we define the right problem to address.
      • 3.1.3 3. Ideate - Expand the problem space by identifying and testing out elements that would circumvent problems.
      • 3.1.4 4. Prototype - inexpensive, scaled down version with the key features to investigate the problem and solution. Goal is to identify the best possible solution.
      • 3.1.5 5. Test - test your prototype with the personas in question.
